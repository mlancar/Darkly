# Directory Traversal

By using the `page` parameter from the `index.php` page, we can exploit it to access other resources such as `/etc/passwd`.  

Using `../../etc/passwd` returns a JavaScript alert with the message "Nope".  
Adding more `../` sequences returns another alert saying "Almost".  

By adding even more `../` sequences, we eventually reach the flag, for example:
